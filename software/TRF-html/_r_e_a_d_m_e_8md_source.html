1 Trans-dimensional random field language model {#mainpage}
2 ======================================
3 See [top-level design](@ref top-level) of the project. \n
4 Here includes the source code of Trans-dimensional random field language model (TRFLM), and the scripts of experiment on several dataset, such PTB corpus.
5 
6 Use `make all` to complie the code and the executables are generated in **bin/**
7 
8 The package is organized as follows:
9 - **src/** includes the C++ source code
10 - **exp/** includes the scripts and data of the language modeling experiments
11  - **exp/Word/** is a word morphology expeirment, a pilot expeirment to modeling English words (character sequences)
12  - **exp/PTB/** is the LM experments on Penn Treebank (PTB) dataset.
13  - **exp/Word/data/** and **exp/PTB/data/** contain all the training, valid and test data for LM training.
14  * **\*.lext** is the vocabulary for ngram LMs;
15  * **\*.list** is the vocabulary used by TRFLMs. The only difference between **\*.lext** and **\*.list** is that **\*.list** removes the begin and end symbols '<s>' and' </s>'. 'c10' or 'c200' in the file name denote the class number is 10 or 200.
16  * **\*.no** is the dataset corresponding to the vocabulary **\*.lext**, used by ngram LMs
17  * **\*.id** is the dataset corresponding to the vocabulary **\*.list**, used by TRFLMs
18  - **exp/Word/ngram/** and **exp/PTB/ngram/** contain the python scripts for ngram LMs
19  - **exp/Word/trf/** and **exp/PTB/trf/** contain the python scripts for TRFLM training
20
